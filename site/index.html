Why CodePrint

Built to be different from every "AI detector" out there.

Most tools return a bare percentage from an opaque text model. CodePrint was designed around the failure modes that make those tools unusable in real classrooms.

Structure, not text

Features come from the abstract syntax tree, token-level style, and statistical predictability — signals that survive variable renaming and comment stripping, where text-based detectors fall apart.

Calibrated abstention

Monte-Carlo dropout gives every verdict an uncertainty estimate. When the model isn't sure, it says INCONCLUSIVE instead of gambling with a student's academic record.

Authorship drift

Each student's earlier work builds a personal stylometric fingerprint. A submission is compared to that student's own history — catching strong students an absolute classifier would miss, and protecting naturally "clean" coders.

Private by architecture

The entire pipeline runs in your browser (or on-premise via the CLI). Student code is never sent to a third-party API — a hard requirement most institutions have and most detectors ignore.

Hearing-ready evidence

Every flag lists the exact signals that drove it — observed value, typical human range, typical AI range, and deviation — in plain English an integrity panel can read.

Honest evaluation

Training splits are grouped by author, so the model is never tested on students it trained on — the data leakage that quietly inflates most published detector accuracy numbers.

How it works

From raw source to defensible verdict.

Fingerprint the code

~100 features are extracted per submission: AST shape and nesting, naming and formatting habits, comment behavior, and burstiness under a course-corpus language model.

Classify with uncertainty

A compact neural network scores the fingerprint 30 times with stochastic dropout. The spread of those scores becomes the verdict's confidence — and its license to abstain.

Compare to the student

The submission's style is checked against the student's own past work. A sudden stylistic jump is flagged as drift, with the shifted features named explicitly.
